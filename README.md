# JavaScript Studio tsconfig.json

> Using the TypeScript compiler with JSDoc

This configuration is supposed to be used with [`@studio/eslint-config`][1].

## Usage:

`tsconfig.json`

```json
{
  "extends": "@studio/tsconfig",
  "include": ["**/*.js"],
  "exclude": ["node_modules"]
}
```
